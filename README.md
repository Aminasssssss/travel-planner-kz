## Tech Stack
- **Frontend:** Angular 17+
- **Backend:** Django + Django REST Framework
- **Auth:** JWT (SimpleJWT)
- **Map:** Leaflet.js

## Run

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```bash
cd frontend
npm install
ng serve
```
